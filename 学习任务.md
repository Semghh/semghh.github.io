# 学习任务



## 1.大一

### 1.1 算法

1. 每周最少2道算法题 ， 留的题目所有人相同。做出来或者已经理解的成员可以分享给没有做出来的同学。

   视个人自身情况额外加题。视成员整体完成情况逐步增加题量。在大三找实习前，要尽量达成150道

2. 留的算法题要分类做题。每一种算法，所有人都要记笔记，留下自己的思考。

3. 有一些优秀的刷题博客网站

   

   https://codetop.cc/login  codeTop

   https://suanfa8.com/video/ 算法8

   https://programmercarl.com/  代码随想录

4. 常用的算法思想包括但不限于

   第一层:  链表 ， 哈希表 ，排序 (快排，归并) ，双指针， 数组 ，字符串 等

   第二层 :  深搜，广搜，二叉树，回溯  等

   第三层:  滑动窗口，贪心，dp 等



```
大一尽量以 简单题为主。

刷算法心得: 尽量分类刷题，一个新的类型应当充分理解刷简单题以后，再尝试中等。进而困难
```





### 1.2 提前修专业课

```
借用高年级成员的书，笔记，提前修一些重要的课程。例如 OS，计网。为后续项目

一些水课就不用听了，带电脑，带书，学习自己的主修课
```













## 2.大二



### 2.1 算法

​	1.每周最少3道，统一留题目  ，视个人自身情况额外加题。视成员整体完成情况逐步增加题量/更改计划



### 2.2 假期

1.利用假期时间学习算法，学习各方向技术  

```
例如，如果是JavaWeb方向，则应当
JavaSE ,数据库 ->
Spring ,Mybatis ->
SpringMVC ,SpringBoot ,Redis->
JVM ,JUC ->
MQ ,SpringCloud ->
...
```



### 2.3 项目



```
在拥有了
JavaSE ,数据库 ->
Spring ,Mybatis ->SpringMVC 
时就可以独立写一个SSM项目了。
这个项目可以作为大三的结课作业。
在上课的时候就可以学其他的技术了。
```



## 2.4 实习

如果个人实力过硬，是可以在大二的暑期实习的。

```
刷面经，刷牛客
```









## 3.大三



### 3.1 持续提高算法能力

此时可能算法水平有较大分层了，按照自己能力为自己定制算法计划。

```
面试三大马车: 项目，技术力，算法
```





### 3.2 为实习做准备

```
提高 OS , 计网能力。
准备面试八股, 写简历，改简历。 刷面经

深挖项目
```



### 3.3 分享经验

团队的中坚力量。

回顾自己的知识，分享自己的学习思路，学习经验。

带领大二，大一成员学习，根据实际情况，指定学习计划

```
一个良好的面试表达能力，是面试成功的关键。
```
