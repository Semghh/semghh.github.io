# 1. 后台管理项目

使用的是 GitHub上的模板 `vue-admin-template`

项目地址 `https://github.com/PanJiachen/vue-admin-template`











## 1.1 项目结构



build文件夹 ---index.js  

```
webpack 配置文件
```



mock 文件夹	

```
数据的文件夹。  模拟一些假的数据
```





node_modules 文件夹

```
项目依赖的模块
```



public 文件夹

```
包含 .ico  和 index.html

经常放置一些静态资源。 Webpack不会编译这个文件夹。原封不动打包到dist
```





src

```
---api  有关请求接口

--assets 静态资源，webpack会编译

---components 非路由组件和全局组件
---icons 矢量图
---store 与vuex相关
---router 与路由相关
---views 放置路由组件

```
